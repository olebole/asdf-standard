CoreÂ¶

The core module contains schema that must be implemented by every asdf library.

  • asdf: Top-level schema for every ASDF file.
  • complex: Complex number value.
  • ndarray: An n-dimensional array.
  • table: A table.
  • column: A column in a table.
  • constant: Specify that a value is a constant.
  • software: Describes a software package.
  • history_entry: An entry in the file history.
